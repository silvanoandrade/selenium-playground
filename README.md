# Selenium Playground 🚕🤖

This repository contains my daily Selenium practice, exercises, and automation testing experiments developed during the **TripleTen QA Bootcamp**.

The main purpose of this repository is to strengthen my understanding of **web automation, UI testing, element interaction, and test validation** using Python and Selenium WebDriver.

Although these are not final portfolio projects yet, this space represents my **hands-on learning process, experimentation, and continuous growth in test automation**.

## 📚 What I'm Practicing
- Web element locators (`ID`, `XPath`, `CSS Selector`)
- Form filling automation
- Button clicks and navigation
- Assertions and validations
- Explicit waits (`WebDriverWait`)
- Dynamic elements
- Pop-ups and overlays
- Basic UI workflow testing
- Selenium debugging practice

This repository documents my evolution in **Selenium automation testing**, helping me build a solid foundation for future real-world QA automation projects.

Every script here represents an important step in my journey toward becoming a **QA Automation Engineer**.
